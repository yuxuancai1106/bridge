## 🚀 Quick Start

```bash
# 1. Setup (one command!)
./setup.sh

# 2. Set up Supabase
# - Create a Supabase project
# - Run the SQL schema: supabase-schema.sql
# - Update .env.local with your credentials

# 3. Start the app
npm run dev

# 4. Open http://localhost:3000
```
